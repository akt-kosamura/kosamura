# 秋高考査村

無料・匿名で使える過去問共有サイト

## 概要

秋高考査村は、学生が過去問を匿名で共有・検索できるWebアプリケーションです。定期考査、実力テスト、確認テストなどの問題をPDFや画像形式でアップロードし、他の学生と共有することができます。

## 主な機能

- **ファイルアップロード**: PDF、画像ファイルのアップロード機能
- **問題検索**: 学年、年度、科目、種類別での問題検索
- **匿名共有**: 完全匿名での過去問共有
- **レスポンシブデザイン**: PC版とスマートフォン版の両方に対応
- **管理機能**: 管理者による投稿管理・削除機能
- **いいね機能**: 投稿への評価システム

## 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **バックエンド**: Google Apps Script (GAS)
- **データベース**: Google Spreadsheet
- **ファイルストレージ**: Google Drive
- **UIライブラリ**: Font Awesome

## セットアップ

### 前提条件

- Google Apps Script プロジェクト
- Google Drive フォルダ
- Google Spreadsheet

### 環境変数の設定

Google Apps Script のスクリプトプロパティで以下の環境変数を設定してください：

- `FOLDER_ID`: アップロードファイル保存用のGoogle DriveフォルダID
- `SPREADSHEET_ID`: メタデータ保存用のGoogle Spreadsheet ID

### デプロイ手順

1. Google Apps Script で新しいプロジェクトを作成
2. `code.gs` の内容をコピー＆ペースト
3. 環境変数を設定
4. Webアプリケーションとしてデプロイ
5. デプロイされたURLを `assets/js/api.js` の `baseURL` に設定

## 使用方法

### ファイルアップロード

1. アップロードページにアクセス
2. 学年、年度、科目、種類を選択
3. ファイルを選択（PDF、画像対応）
4. コメントを入力（任意）
5. アップロード実行

### 問題検索

1. 検索ページにアクセス
2. 検索条件を設定（学年、年度、科目など）
3. 検索実行
4. 結果から目的のファイルを閲覧

### 管理機能

- 管理者パスワード: `0611`
- 投稿の削除、編集が可能
- 一括削除機能

## ファイル形式対応

## 開発者向け情報

### API エンドポイント

- `getData`: データ取得
- `uploadFileAndRecord`: ファイルアップロード
- `like/unlike`: いいね機能
- `bad/unbad`: 悪い評価機能
- `deletePost`: 投稿削除
- `updatePost`: 投稿更新

### カスタマイズ

- CSSファイルを編集してデザインを変更
- `api.js` でAPI通信の動作をカスタマイズ
- `code.gs` でバックエンド処理を拡張
